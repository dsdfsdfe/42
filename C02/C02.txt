strcpy(a, b) : b문자열을 a에 복사

1. strcpy 함수가 복제 역할이니까 dest=src로 만들어서 dest를 출력해야함
2. Int I 선언 / i를 0으로 정의하고(첫구간부터 시작해야하니까) src가 null값이 아닐때동안 반복문을 생성
3. dest[I] = src[I] 로 한구간씩 올려가며(++i) 값을 같게 만들어준다
4. Dest에 null값을 붙여주고 dest를 리턴한다

<Ex01>

strncpy(a, b, n) : b문자열을 a에 n길이만큼 복사

1. unsigned int i 선언 / i를 0으로 정의하고(첫구간부터 시작해야하니까) src가 null값이 아닐때 + i가 n보다 작을동안 반복문을 생성
2. dest[I] = src[I] 로 한구간씩 올려가며(++i) 값을 같게 만들어준다
3. I가 n보다 작을때, Dest에 null값을 붙여주고 i에 1을 추가
4. dest를 리턴한다
